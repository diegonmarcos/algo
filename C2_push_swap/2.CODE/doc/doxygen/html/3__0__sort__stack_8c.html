Detailed Description

Implements the main sorting logic for the push_swap program.

Details:
This module selects and applies the appropriate sorting algorithm based on the size of stack A:
  • Selection Sort of sorting algorithm for below 50 elements
  • Radix for 50 to 65 elements
  • Quick Sort for 65 to 75 elements
  • Gear Sort for 75 to 100 elements
  • Gear Sort for 100 to 500 elements
  • Optimization of moves after sorting:
    • Replace with double rotate or double reverse rotate when possible
    • Remove unnecessary moves(canceling moves)
Functions:
  • void sort_stack(t_list_program *list_program): Main function to sort the stack using the appropriate algorithm.
  • void selection_sort(t_list_program *list_program): Sorts the stack using the selection sort algorithm.

  • void gear_sort_pushswap(t_list_program *list_program): Sorts the stack using the gear sort algorithm.
  • void sort_stack_dbg(t_list_program *list_program): Debug function for sorting the stack.
  • void moves_optimizations(t_list_program *list_program): Optimizes the moves after sorting.
