Implementation of the push\+\_\+swap project from 42 school. 

\begin{DoxyParagraph}{Details\+:}

\begin{DoxyItemize}
\item The push\+\_\+swap project is a sorting algorithm project that involves sorting a two stack of integers using stacks operations. The goal is to sort the stack with the least number of operations possible. The allowed operations are pushing elements between two stacks, rotating or reversing the stacks, and swapping the top elements of the stacks.
