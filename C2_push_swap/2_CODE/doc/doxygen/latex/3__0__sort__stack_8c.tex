\doxysubsection{Detailed Description}
Implements the main sorting logic for the push\+\_\+swap program. 

\begin{DoxyParagraph}{Details\+:}
This module selects and applies the appropriate sorting algorithm based on the size of stack A\+:
\end{DoxyParagraph}

\begin{DoxyItemize}
\item Selection Sort of sorting algorithm for below 50 elements
\item Radix for 50 to 65 elements
\item Quick Sort for 65 to 75 elements
\item Gear Sort for 75 to 100 elements
\item Gear Sort for 100 to 500 elements
\item Optimization of moves after sorting\+:
\begin{DoxyItemize}
\item Replace with double rotate or double reverse rotate when possible
\item Remove unnecessary moves(canceling moves)
\end{DoxyItemize}
\end{DoxyItemize}
